Dive'in

Date : novembre 2022.

Durée : 2 jours.

Description : Projet universitaire : site web statique réalise en HTML / CSS.

Projet :

Ce projet a été réalisé dans le cadre des SAEs S1.05 et S1.06, par groupe de 3. Le but était de créer un site web statique, pour une entreprise fictive basée sur l'évènementiel.

Nous devions utiliser uniquement du HTML et du CSS, avec Visual Studio Code. Nous avons mis en place différents éléments clés des pages web : barre de navigation, footer, header, articles, sections..
