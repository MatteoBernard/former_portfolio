SkylandersWiki

Date : avril 2024.

Durée : ~15 heures

Description : Projet personnel : application mobile hybride, wiki sur les figurines Skylanders.

Projet :

J'ai réalisé ce projet pour m'entraîner à la création d'applications mobiles hybrides, mais surtout pour me découvrir le framework React Native, et pour renflouer mes connaissances en Typescript.

L'application est un wiki sur les figurines Skylanders, elle permet de consulter les différents personnages, éléments, jeux et figurines de la licence.

J'ai utilisé une API que j'ai également réalisée, plus d'informations ici. Les appels API se font à chaque chargement d'une nouvelle page. Il y a également un système de filtrage en fonction de différents paramètres.
