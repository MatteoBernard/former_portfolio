JADex

Date : novembre 2023 (en cours).

Durée : 4 mois (1h30 par semaine).

Description : Projet universitaire : pages web dynamiques avec capture fictive de Pokémons, fait en Python / Flask, et utilisation de l'API Pokémon.

Projet :

Ce projet a été réalisé dans le cadre du cours de MSI (Management des systèmes d'information). Le but était de créer un projet, en respectant les différentes étapes de développement et de gestion de projet : dailys, sprints, comptes rendus, issues, issue boards...

Le projet a été réalisé en Python / Flask, afin de gérer facilement les pages web. Nous avons également utilisé une base de données sqlite pour stocker les Pokémons attrapés, ainsi que différents outils front : Bulma, Bootstrap et Figma.
