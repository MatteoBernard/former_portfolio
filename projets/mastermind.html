Mastermind réseau

Date : novembre 2023.

Durée : 4h30.

Description : Projet universitaire : reprise du jeu "Mastermind" en Java, avec communication client / serveur, version TCP.

Projet :

Ce projet a été réalisé dans le cadre du cours de "Programmation système". Le but était de coder un client et un serveur, et les faire communiquer.

Le client et le serveur sont codés en Java, en faisant hériter les deux classes de la classe Thread. Le jeu se déroule de la sorte : le client entre un nombre d'essais maximum, puis il tente de deviner la suite de chiffres générée. Si le chiffre se trouve à la bonne place dans la suite, le serveur lui renvoie le nombre. Si le chiffre se trouve dans la suite, mais à la mauvaise place, le serveur renvoie un "X". Enfin, si le chiffre n'est pas dans la suite, le serveur renvoie un "-".
