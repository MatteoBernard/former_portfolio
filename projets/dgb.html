DawgsGatitosAndBirdies

Date : avril 2024.

Durée : ~10 heures

Description : Projet personnel : application mobile hybride, afficheur d'images d'animaux.

Projet :

J'ai réalisé ce projet afin de m'entraîner sur une technologie vue en cours : Ionic. J'ai utilisé également du Typescript, du CSS et du Angular. Une version Android est disponible en release sur Github.

L'application permet d'afficher des images d'animaux mignons, de manière aléatoire en faisant des appels sur cette API. Les animaux affichables sont les chats, les oiseaux et les chiens de race shiba.
