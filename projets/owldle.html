OWLdle

Date : septembre 2022.

Durée : 1 semaine.

Description : Projet perso : application en JavaFX, jeu type "Wordle" sur le jeu Overwatch et son esport. Selon les modes, le joueur doit trouver quel est le héros, ou quel est le joueur professionnel d'Overwatch, basé sur des indices.

Projet :

J'apprécie particulièrement la scène Overwatch. Malheureusement, celle-ci est beaucoup moins vivante que celle d'autres jeux comme League of Legends, ou Valorant par exemple. J'ai donc décidé de participer, à mon échelle, à la vie de cette communauté, en créant un petit jeu sur ordinateur.

J'ai réalisé ce projet en Java / JavaFX, afin d'en faire une application de bureau exécutable. Je me suis aidé de SceneBuilder pour construire les vues, et ai stylisé le tout avec du CSS.

Concernant le stockage des données à deviner, je suis passé par une base de donnée en ligne, MongoDB, afin d'éviter de devoir re-sortir une version du jeu à chaque transfert de joueur / ajout de nouveau héros.

Le projet se compose pour l'instant de trois vues : une pour l'accueil, et une pour chaque mode de jeu.
