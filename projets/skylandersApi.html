SkylandersApi

Date : avril 2024.

Durée : ~10 heures

Description : Projet personnel : API Rest sur les figurines Skylanders.

Projet :

J'ai réalisé ce projet pour pouvoir replonger dans une licence de mon enfance, les Skylanders. Cela m'a également permis de découvrir le framework Express, et de m'améliorer en Javascript.

L'API permet de récupérer des informations sur les figurines, personnages, jeux et éléments de la licence Skylanders. Les routes renvoient des données au format JSON. On retrouve également une documentation Swagger.
Le lien de l'API : SkylandersApi
