FlappyCapy

Date : octobre 2023.

Durée : 1 semaine.

Description : Projet perso : Reprise du jeu Flappy Bird, fait en Python / Pygame. Le joueur doit éviter les obstacles afin de réaliser le meilleur score possible.

Projet :

J'aime beaucoup les capybaras, et je trouvais cela fun de créer un petit jeu autour d'eux. J'ai donc créé une reprise du jeu "Flappy Bird", avec un capybara voyageant dans l'espace, sur sa fusée.

J'ai réalisé ce projet en Python / Pygame, afin de découvrir la bibliothèque Pygame. Pour les sprites, je les ai trouvés sur Freepik, et modifiés sur Gimp. Le meilleur score du joueur se stocke et s'incrémente automatiquement dans un fichier au format JSON.
