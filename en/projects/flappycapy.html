FlappyCapy

Date : october 2023.

Duration: 1 week.

Description: Personal project: resumption of the game Flappy Bird, made in Python / Pygame. The player must avoid obstacles in order to achieve the best possible score.

Project :

I really like capybaras, and I thought it would be fun to create a little game around them. So I have created a resumption of the game "Flappy Bird", with a capybara traveling in space, on his rocket.

I made this project in Python / Pygame, in order to discover the Pygame library. For the sprites, I found them on Freepik, and modified them on Gimp. Player's best score is stored and automatically incremented in a JSON format file.
