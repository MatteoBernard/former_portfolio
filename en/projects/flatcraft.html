Flatcraft

Date : november 2022 (ongoing).

Duration: 4 months (3 hours per week).

Description: University project: Resumption of the game "Minecraft" in 2D, in Java.

Project :

This project was made as part of the QDev (Development Quality) course. The goal is to create a game inspired by "Minecraft", all in 2D. There are three dimensions of Minecraft, an inventory, blocks to mine and use for different crafts, as well as different biomes and terrain structures.

The project was made in Java, and the display is done in JavaFX. The implementation of design patterns was central: abstract factories, states...
