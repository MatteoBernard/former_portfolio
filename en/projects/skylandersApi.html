SkylandersApi

Date : april 2024.

Duration: ~10 hours

Description: Personal project : Rest API on Skylanders figures.

Project :

I carried out this project to be able to dive back into a license from my childhood, Skylanders. This also allowed me to discover the Express framework, and to improve my skills in Javascript.

The API allows you to retrieve information on figurines, characters, games and elements of the Skylanders license. Routes return data in JSON format. There is also Swagger documentation.
API link: SkylandersApi
