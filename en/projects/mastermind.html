Network Mastermind

Date : november 2023.

Duration: 4h30.

Description: University project: resumption of the game "Mastermind" in Java, with client/server communication, TCP version.

Project :

This project was made as part of the "System Programming" course. The goal was to code a client and a server, and make them communicate.

The client and server are coded in Java, inheriting both classes from the Thread class. The game goes like this: the customer enters a maximum number of tries, then tries to guess the sequence of numbers generated. If the number is in the correct place in the sequence, the server returns the number. If the number is in the following, but in the wrong place, the server returns an "X". Finally, if the figure is not not in the following, the server returns a "-".
