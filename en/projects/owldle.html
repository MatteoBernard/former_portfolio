OWLdle

Date : september 2023.

Duration: 1 week.

Description: Personal project: application in JavaFX, “Wordle” like game on the Overwatch game and its esport. Depending on the mode, the player must find who is the hero, or who is the professional Overwatch player, based on clues.

Project :

I particularly enjoy the Overwatch scene. Unfortunately, this one is much less alive than that of other games like League of Legends, or Valorant for example. So I have decided to participate, on my own scale, in the life of this community, by creating a small computer game.

I created this project in Java / JavaFX, in order to make it an executable desktop application. I used SceneBuilder to build the views, and styled everything with CSS.

Concerning the storage of the data to be guessed, I used an online database, MongoDB, in order to avoid having to re-release a version of the game each time a player transfers / adds a new hero.

The project currently consists of three views: one for the reception, and one for each game mode.
