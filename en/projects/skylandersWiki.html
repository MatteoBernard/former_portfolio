SkylandersWiki

Date : april 2024.

Duration: ~15 hours.

Description: Personal project: hybrid mobile app, Skylanders action figure wiki.

Project :

I carried out this project to train myself in creating hybrid mobile applications, but above all to discover the React Native framework, and to refresh my knowledge of Typescript.

The application is a wiki on Skylanders figurines, it allows you to consult the different characters, elements, games and figurines of the license.

I used an API that I also created, more information here. API calls are made each time a new page is loaded. There is also a filtering system based on different parameters.
